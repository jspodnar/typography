TypeSet 4.0 Sample Page

What is TypeSet?

TypeSet is a JavaScript Object that automatically typesets your page for ideal font size, leading, and line length. It's intent is to set base body copy configuration that maximizes readibility, while at the same time allows you to override as required.

How does it do that?

  1. First: It determines your screen resolution.
  2. From there, it determines what an ideal Field of View is for 1–4 characters, and set the font size accordingly.
  3. At the same time, it will also determine the default font size your device has set. If your font size is larger than the calculation TypeSet has determined, it will your your preference instead.

From there, TypeSet will calculate ideal length and leading based on the base body copy size, and set that also.

How do I use it?

Apply the "typeset" class to the body tag to apply typesetting to your page. Override TypeSet settings with your own CSS as required.
